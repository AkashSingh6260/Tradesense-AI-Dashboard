Section 01

How the Website Works

TradeSense AI is a React-based, client-side web application designed as an AI-powered stock market dashboard for Indian markets. It currently uses static mock data to demonstrate the full user experience without requiring a live API connection.

📁 Project Structure

Folder / File Purpose
App.js Root component — defines all page routes using React Router
components/ Reusable UI pieces: Navbar, StockCard, AIRecommendationCard, etc.
pages/ The 5 main application pages
context/ Global state management: ThemeContext, UserContext
landing_page/ Marketing homepage split into sub-sections (Hero, About, Pricing…)
utils/mockData.js All simulated stock data (indices, holdings, AI signals)
styles/globals.css CSS variables for the entire design system (colors, radius, shadows)

🗺️ Routing Overview

Route Page Description
/ Landing Page Marketing homepage with "Open App" CTA
/dashboard Dashboard Portfolio summary, market overview, AI signals
/markets Markets All stocks, indices, sector performance
/portfolio Portfolio Holdings table, P&L charts, sector allocation
/watchlist Watchlist Searchable list of tracked stocks with AI signals
/analysis Stock Analysis Deep dive: price chart, RSI gauge, AI recommendation

🎨 Theme System

The app supports Dark and Light modes via ThemeContext.js. The selected theme is saved to localStorage and applied by toggling a data-theme attribute on the <body>. All colors are CSS variables (e.g., --green-buy, --red-sell, --bg-card) that automatically switch based on the active theme.

⚠️ Data Note

All stock data is currently static mock data from utils/mockData.js. The backend folder is empty and no real API is connected yet. Portfolio history and price charts are randomly generated on each page load using JavaScript functions.

🧠 The Core Differentiator — The AI Interpretation Layer

❌ Existing Platforms

  • Show RSI as a raw number — user must know what it means
  • Show Moving Averages on a chart — user must interpret crossovers
  • Provide charts and data, but no decision support
  • Require financial literacy to use effectively
  • Designed for experienced traders

✅ TradeSense AI

  • RSI shown as a visual arc gauge with label: "Overbought / Neutral / Oversold"
  • AI translates chart patterns into a Buy / Hold / Sell decision
  • Provides a confidence percentage for each signal
  • Writes a plain-English insight explaining the reasoning
  • Designed for students and retail beginners
🔑 Key Innovation

No free platform currently provides plain-English AI insights for retail investors in India. Bloomberg Terminal does this for professionals at ₹16 lakh/year. TradeSense AI democratizes this capability for students and first-time investors — bridging the gap between raw data and actionable decisions.
